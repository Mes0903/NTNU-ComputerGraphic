3D Scene with Environment Cube Mapping

Project Features:

  • Environment Cube Mapping: Realistic skybox using Yokohama environment textures
  • Cube Map Reflections: Objects reflect the surrounding environment
  • Phong Lighting Model: Ambient, diffuse, and specular lighting with adjustable parameters
  • Texture Mapping: High-quality texture applied to 3D models
  • Dual Camera System: Switch between first-person and third-person views
  • Interactive Controls: Real-time adjustment of lighting and reflection parameters

How to Use:

  • Use WASD keys to move around the scene
  • Move your mouse to look around (automatic in first-person, click-drag in third-person)
  • Toggle between first-person and third-person view using the checkbox
  • Adjust rendering modes to see different visual effects
  • Modify lighting and reflection settings using the control panel

Technical Implementation:

  • WebGL with custom shaders for rendering
  • Cube map texture loading and sampling
  • Real-time reflection vector calculation
  • Phong lighting model implementation
  • Interactive camera controls with pointer lock API
